HTML

  • HTML elements create structure to a webpage.
  • HTML elements are composed of an opening and closing tag enclosed in angled brackets.
  • The head element contains information about the webpage.
  • Metadata is data about our webpage that the browser uses to serve webpage users.
  • The body element represents the visible content shown to the user.
  • A heading element ranges from 1-6; h1 being the largest and only used once for a main title.
  • ordered and unordered lists can be created using their respective elements.
  • Image elements contain information about images on the webpage that can contain attributes, most importantly the source attribute.
  • Anchor elements create links to webpages.
  • ID and class attributes identifies elements to be selected for styling.

CSS

  • CSS adds personalization to webpages.
  • The CSS file needs to be linked into the HTML file.
  • elements and attributes can be selected to personalize.
  • personalization can include: color, background-color, box-shadow, text-alignment, font-size, and much more!
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout [branch-name]: switches branches.
  • git commit -m[message]: Commits staged changes.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements can be created using JavaScript with if and else if statement.s
  • An Array is a single variable that stores a group of related data.
  • Arrays can store strings, numbers, and boolean values.
  • Strings must be contained in quotation marks.
  • Arrays have indices (location identifiers) that begin at zero.
  • Loops can be created to log the values in the array.
  • For loops contains three statements: the starting point for the loops, the condition of the loop, and how the loop presents its data.
  • A function is a set of instructions tha tells the computer how to preform a task.
  • Functions must be called in orfer to execute.
